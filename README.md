# Claude Code Skills

Custom skills for Claude Code focused on .NET/Blazor development.

## Package Versions

| Package | Version |
|---------|---------|
| KnockOff | 10.4.1 |
| Neatoo | 10.3.1 |

## Skills

| Skill | Description |
|-------|-------------|
| **bunit** | Blazor component testing with bUnit. Testing user interactions, mocking services, testing MudBlazor components. |
| **docs-snippets** | Documentation snippet synchronization. Keeps code examples in docs compiled and tested using `#region docs:*` markers. |
| **knockoff** | KnockOff source-generated test stubs. Type-safe interface stubs for unit tests as a Moq alternative. |
| **mudblazor** | MudBlazor component library for Blazor. Forms, data grids, dialogs, navigation, theming, and Neatoo integration. |
| **playwright-blazor-e2e** | End-to-end testing for Blazor with Playwright. Handles Blazor WebAssembly loading, MudBlazor components, and async rendering. |
| **questpdf** | QuestPDF report generation. Fluent C# API for PDF documents, invoices, data tables, and reusable components. |
| **roslyn-source-generators** | Roslyn source generator development. Incremental generators, ForAttributeWithMetadataName patterns, testing strategies. |

## Usage

Skills are automatically invoked by Claude Code when relevant to the task. Each skill provides specialized knowledge and patterns for its domain.
